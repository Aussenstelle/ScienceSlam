Software

Ein Testprojekt zur überprüfung der TCP verbindung ist fertig und die Tests können in kürze beginnen. Das Script wurde in Python geschrieben. In dieser Sprache wird später das GUI (Graphical User Interface) mit den Modulen tkinter, os, time, sys, dir, PIL (Pillow) und zur Nezwerkkomunikation wird das Standatmodul sockets verwendent.

Über das Projekt

Das Projekt soll eine SmartHome komunikation darstellen. Der Netzwerkverkehr funktioniert mit MorseBinNet 2. Die GUI, mit der man das System steuern kann, mit Python 3.

    Das Projekt besteht aus folgenden Komponenten:
  • dem Gateway
  • der Bridge
  • Client bzw. Server
